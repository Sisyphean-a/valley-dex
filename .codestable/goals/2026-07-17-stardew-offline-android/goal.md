# 星露谷离线图鉴 Android 交付

## 目标

从零实现可安装的原生 Android 离线图鉴。应用必须真实安装和查询 schema 2 的 `.svdata`，而非使用硬编码或演示数据。

## 起点

仓库已有完整产品规范、数据库消费契约和 `database/stardew-zh-cn.svdata`。当前没有 Android 工程；本机有 Android SDK、Android Studio JBR 与 ADB，暂无已连接设备。

## 验收标准

- 可安全导入、验证、激活、回滚真实数据包，且内容库以只读方式打开。
- 可离线搜索、分类浏览、查看主要实体详情和本地图片。
- 收藏、历史、笔记、设置与数据包内容分库保存，升级不丢失。
- Debug/Release 构建、单元测试、Lint 与可用的仪器测试均有实际证据；生成可安装 APK。

## 非目标

不接入网络、账号、广告、云同步、Wiki 抓取或玩家存档；不改写内容数据库；不提交真实游戏数据。

## 决策与假设

- 使用用户提供的真实 `.svdata` 做本地开发验证，并只作为被忽略的可选内置资源复制到构建产物。
- Release 使用本机 debug 签名生成可安装的验证 APK；正式发布仍须替换为私有发布签名。
- 不把任何未知 `extra_json` 字段解释成游戏结论，统一按字段存在性降级展示。

## 当前状态

已完整读取产品规范和数据库参考文档，确认数据包为 schema 2、3688 个实体。界面设计先于 Compose 实现落盘。

## 下一步

建立 Gradle 工程与设计系统，随后实现数据包安装和启动链路。
